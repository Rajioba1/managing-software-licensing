# Claude Usage

This repository includes `SKILL.md`, which is operational guidance for software licensing compliance (not legal advice).

## Recommended Use

1. Add `SKILL.md` to your Claude Project Knowledge, or paste it into a session.
2. Ask a specific task so Claude can follow the checklist and templates.

## Example Prompt

```
Use the managing-software-licensing guide to review my dependency licenses and produce an attribution checklist.
```
