# MPL 2.0 FAQ (Secondary Licenses)

Source: https://www.mozilla.org/en-US/MPL/2.0/FAQ/

- MPL 2.0 permits “Secondary Licenses” to allow GPL/LGPL/AGPL re-licensing.
- Secondary License incompatibility is opt-in via Exhibit B ("Incompatible With Secondary Licenses").
- Whether Secondary Licenses apply can depend on file headers or project notices.
- If Secondary License terms are not granted, MPL code remains MPL-only.

Use the official FAQ for precise guidance and examples.
