# Choose A License (Guidance)

Source: https://choosealicense.com/

- Provides a simple decision flow for picking a common license.
- Recommends matching community norms when contributing to an existing ecosystem.
- Highlights common choices:
  - MIT for permissive simplicity.
  - GPL-3.0 for strong copyleft and sharing improvements.
- Includes guidance on what happens if a project has no license.

Use this reference for quick, user-friendly framing, not as a substitute for full license texts.
