# GPLv3 Written Offer / Source Availability (Section 6)

Source: https://www.gnu.org/licenses/gpl-3.0.en.html

- GPLv3 Section 6 allows conveying object code if you provide the Corresponding Source in permitted ways.
- If you provide a written offer, it must be valid for at least three years and as long as you offer spare parts or customer support for that product model.
- The offer must be for anyone who possesses the object code, at no more than the cost of physically performing source distribution.

Use the official GPLv3 text for exact obligations and wording.
