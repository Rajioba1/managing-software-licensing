# GPLv2 Written Offer (Section 3)

Source: https://www.gnu.org/licenses/gpl2.html

- GPLv2 Section 3 allows object code distribution if you do one of: ship source, provide a written offer, or pass through a received offer (noncommercial only).
- The written offer must be valid for at least three years and available to any third party.
- The offer may charge no more than the cost of physically performing source distribution.
- "Complete corresponding source" includes all modules, interface definition files, and build/installation scripts, with OS components excluded unless shipped with the executable.

Use the official GPLv2 text for exact obligations and wording.
